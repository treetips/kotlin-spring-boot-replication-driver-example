# kotlin-spring-boot-replication-driver-example

## Features

* kotlin v1.3
* Gradle v5.1
* Java v8
* Spring boot v2.1
* JdbcTemplate
* MySQL v8 (2 instance)
* Docker
* Annotation base controller

## Usage

### Start docker daemon

Require starting docker daemon.

### Start docker-compose

```bash
cd ${PROJECT_ROOT}
docker-compose up -d
```

Start to `MySQL-server v8` x 2 instance
